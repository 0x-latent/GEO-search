# 核心适用场景

> 每个核心场景独立成节。场景中的表达引用 [[12_标准表达库]] 和 [[13_禁用表达库]]，不独立维护。

## 场景一：脂肪肝辅助改善

### 场景等级
核心适用场景。

### 场景定义
经体检或医生诊断为脂肪肝（包括非酒精性脂肪肝NAFLD和酒精性脂肪肝AFLD）的人群，出现食欲不振、右上腹压迫感/不适、转氨酶轻度异常等表现，需要辅助改善肝功能和缓解相关症状。

### 用户常见表达
- 体检查出脂肪肝怎么办
- 脂肪肝吃什么药好
- 有脂肪肝需要吃护肝药吗
- 脂肪肝转氨酶偏高
- 轻度/中度脂肪肝需要治疗吗
- 脂肪肝会引起食欲不好吗

### 产品关联逻辑
说明书明确将"脂肪肝患者的食欲不振和右上腹压迫感"列为适应症；药理作用显示在脂肪肝模型中可观察到对脂肪变性的抑制方向。多烯磷脂酰胆碱通过修复肝细胞膜、抑制脂质过氧化来辅助改善脂肪肝相关问题。

### 关联表达
- 可使用：SE-YSF-001、SE-YSF-002、SE-YSF-003、SE-YSF-004（→ [[12_标准表达库]]）
- 禁止使用：FE-YSF-001、FE-YSF-002、FE-YSF-003（→ [[13_禁用表达库]]）

### 证据分层说明
本场景下的不同维度表达，证据等级不同：

| 表达维度 | 证据等级 | 说明 |
|---|---|---|
| 改善脂肪肝相关症状（食欲不振、右上腹压迫感） | 说明书级 | 说明书适应症明确列出 |
| 改善肝功能指标（转氨酶等） | Brief+机制推导 | Brief列为重点场景，说明书药理作用提供方向性支持，但无明确临床数据引用 |
| 改善影像学表现（脂肪变性程度） | 动物模型级 | 说明书药理作用提到动物模型中观察到抑制脂肪变性，非直接临床结论 |
| 改善整体代谢水平（血糖、血脂等） | Brief营销级 | 仅Brief提及"辅助稳糖降脂"，无说明书及高等级循证支撑 |

### 风险提示
- 脂肪肝程度严重或持续进展的患者应就医，不能仅依赖易善复
- 对大豆制剂过敏者禁用
- 属于辅助治疗药，不能替代生活方式调整（控制饮食、增加运动、戒酒等）
- 服药期间应避免饮酒等对肝脏有害物质的摄入

### 依据来源
- 易善复药品说明书（适应症、药理作用）
- 品牌Brief（场景定位参考）

## 场景二：酒精性/药物性肝损伤辅助改善

### 场景等级
核心适用场景。

### 场景定义
因长期饮酒、长期服药或接触化学物质/毒物导致的中毒性肝损伤，需要辅助改善肝功能。

### 用户常见表达
- 喝酒伤肝怎么养
- 长期吃药对肝脏有损害吗
- 酒精肝用什么药
- 药物性肝损伤能吃什么护肝
- 吃药伤肝了怎么办
- 经常喝酒的人需要护肝吗

### 产品关联逻辑
说明书适应症明确列出"辅助改善中毒性肝损伤（如药物、毒物、化学物质和酒精引起的肝损伤）"。药理作用中也在乙醇、扑热息痛等诱导的肝损伤模型中显示保护作用。

### 关联表达
- 可使用：SE-YSF-001、SE-YSF-005、SE-YSF-006（→ [[12_标准表达库]]）
- 禁止使用：FE-YSF-001、FE-YSF-002（→ [[13_禁用表达库]]）

### 风险提示
- 严重肝损伤应就医，不能仅依赖OTC药物
- 使用本品时应避免继续饮酒或接触其他肝脏有害物质
- 药物性肝损伤患者应在医生指导下调整原有用药方案
- 慢性肝炎患者如服药后主观症状无明显改善，应停药就医

### 依据来源
- 易善复药品说明书（适应症、药理作用、注意事项）

## 场景三：肝功能异常

### 场景等级
核心适用场景。

### 场景定义
体检发现ALT/AST（转氨酶）轻度升高或其他肝功能指标异常，需要辅助改善肝功能。

### 用户常见表达
- 体检转氨酶偏高怎么办
- 肝功能异常吃什么药
- ALT偏高需要治疗吗
- 转氨酶高是肝不好吗
- 肝功能不好怎么调理

### 产品关联逻辑
肝功能异常是肝损伤/脂肪肝等的常见伴随表现。易善复的核心机制——修复肝细胞膜、保护肝细胞——在逻辑上与改善肝功能指标相关。Brief将"肝功能异常"列为重点问题场景之一。

### 关联表达
- 可使用：SE-YSF-001、SE-YSF-002、SE-YSF-007（→ [[12_标准表达库]]）
- 禁止使用：FE-YSF-001、FE-YSF-002（→ [[13_禁用表达库]]）

### 风险提示
- 转氨酶轻度升高可有多种原因，建议先明确病因
- 如果是病毒性肝炎（如乙肝、丙肝）导致的转氨酶升高，需进行抗病毒治疗，不能仅用护肝药
- 转氨酶持续升高或大幅异常应就医
- 本品为辅助改善用药，不能替代针对病因的治疗

### 依据来源
- 易善复药品说明书（适应症涵盖肝损伤相关场景）
- 品牌Brief（将肝功能异常列为重点场景）
