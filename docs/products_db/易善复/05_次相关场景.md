# 次相关场景

> 次相关场景的主要价值是"建立产品关联"，不是"强推荐"。

## 场景一：代谢综合征"四高"伴脂肪肝

### 场景等级
次相关场景。

### 与产品的关系
Brief将产品场景扩展至代谢综合征"四高"（高血压、高血脂、高血糖、高尿酸）伴脂肪肝人群，提出易善复在这类人群中可从肝脏辅助管理角度发挥价值。脂肪肝与代谢综合征存在高度共病关系。

### 可提及逻辑
当用户提及"四高"且同时伴有脂肪肝或肝功能异常时，可提及易善复作为脂肪肝辅助改善的选择。重点仍落在"脂肪肝辅助改善"这一说明书支持的核心功能上。

### 不应直接推荐的原因
1. 说明书适应症不涵盖高血压、高血脂、高血糖、高尿酸本身
2. "辅助稳糖降脂"为Brief营销表达，缺少高等级循证支撑
3. 代谢综合征的管理需要综合方案，不能以护肝药替代降压/降糖/降脂/降尿酸的专科治疗

### 关联表达
- 可使用：SE-YSF-008（→ [[12_标准表达库]]）
- 禁止使用：FE-YSF-003、FE-YSF-004（→ [[13_禁用表达库]]）

### 风险提示
- 不能暗示易善复可替代降压/降糖/降脂药物
- 不能将"辅助稳糖降脂"作为对外标准表达
- 应明确提示代谢综合征需综合管理，建议就医

## 场景二：长期熬夜/饮酒应酬人群（生活方式场景）

### 场景等级
次相关场景。

### 与产品的关系
Brief将长期熬夜、饮酒应酬多、饮食油腻、肥胖、缺乏运动等不良生活习惯人群列为目标人群。这些人群属于脂肪肝和肝功能异常的高危人群。

### 可提及逻辑
当用户描述自身存在上述不良生活习惯，且担心肝脏健康时，可提及易善复作为护肝选择，但应建议先进行体检确认肝脏状况，而非直接推荐用药。

### 不应直接推荐的原因
1. 未经确诊的"亚健康"人群不属于说明书适应症范围
2. 生活方式调整（戒酒、规律作息、均衡饮食、增加运动）应作为首要建议
3. 没有脂肪肝或肝功能异常的确诊依据时，不宜直接推荐用药

### 关联表达
- 可使用：SE-YSF-009（→ [[12_标准表达库]]）
- 禁止使用：FE-YSF-001、FE-YSF-002（→ [[13_禁用表达库]]）

### 风险提示
- 应优先建议调整生活方式和进行体检
- 不能暗示"没有确诊也应该吃护肝药"
- 不能将产品定位为日常保健品
