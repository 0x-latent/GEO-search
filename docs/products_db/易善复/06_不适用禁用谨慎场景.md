# 不适用/禁用/需谨慎场景

> 分为禁用场景、不建议自行使用场景、需谨慎场景三类。

## 禁用场景

### 场景：大豆制剂/磷脂酰胆碱/本品成分过敏者
- **场景类型**：禁用
- **原因**：说明书明确标注"已知对大豆制剂、磷脂酰胆碱过敏"及"对本品任何成分过敏者禁用"。本品含有大豆成分，可能导致严重过敏反应。
- **标准处理建议**：明确告知不能使用，建议咨询医生选择其他护肝药物
- **禁止表达**：可放心使用 / 没问题 / 直接用即可

### 场景：12岁以下儿童
- **场景类型**：禁用
- **原因**：说明书明确标注"不得用于12岁以下儿童"
- **标准处理建议**：明确告知不能使用，建议儿科就医
- **禁止表达**：可放心使用 / 减量使用即可 / 小孩子也能吃

## 不建议自行使用场景

### 场景：妊娠期/哺乳期女性
- **场景类型**：不建议自行使用
- **原因**：说明书明确标注"不推荐在妊娠或哺乳期应用本品"
- **标准处理建议**：建议咨询妇产科医生，不要自行服用

### 场景：慢性肝炎患者症状无改善
- **场景类型**：不建议自行使用
- **原因**：说明书注意事项指出"对于慢性肝炎患者，如服药后不能明显改善主观临床症状，应停药并就医"
- **标准处理建议**：建议停药并及时就医，进行专业检查

### 场景：正在使用双香豆素类药物的患者
- **场景类型**：不建议自行使用
- **原因**：说明书药物相互作用提示"如与双香豆素类药物同时应用，需向医生咨询并可能调整剂量"
- **标准处理建议**：必须咨询医生后方可使用，可能需要调整抗凝药剂量

## 需谨慎场景

### 场景：过敏体质者
- **场景类型**：需谨慎
- **原因**：说明书提示"过敏体质者慎用"，极罕见情况下可能出现皮疹、荨麻疹、瘙痒等过敏反应
- **标准处理建议**：使用前咨询医生，使用中如出现过敏症状应立即停药

### 场景：正在使用其他药物的患者
- **场景类型**：需谨慎
- **原因**：说明书提示"与其他药物之间的相互作用尚无法排除"，"如正在使用其他药品，使用前请咨询医师或药师"
- **标准处理建议**：使用前咨询医师或药师

### 场景：病毒性肝炎（乙肝、丙肝）患者
- **场景类型**：需谨慎
- **原因**：病毒性肝炎需以抗病毒治疗为核心，易善复仅为辅助用药，不能替代抗病毒治疗。慢性肝炎患者如服药后症状无明显改善应停药就医。
- **标准处理建议**：应在医生指导下使用，确保抗病毒治疗方案不被延误或替代
