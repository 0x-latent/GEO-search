# 合规与风险提示

## 禁忌

1. 已知对大豆制剂、磷脂酰胆碱过敏者禁用
2. 对本品任何成分过敏者禁用
3. 不得用于12岁以下儿童

## 注意事项

1. 本品为辅助治疗药，第一次使用前应咨询医师，治疗期间应定期到医院检查
2. 由于含有大豆成分，本品可能会导致严重过敏反应
3. 使用本品时应避免对肝脏有害物质（如酒精等）的摄入
4. 对于慢性肝炎患者，如服药后不能明显改善主观临床症状，应停药并就医
5. 如果症状加重或出现新症状，可能是疾病恶化的征兆，应立即就医
6. 应严格按推荐剂量服用，不得超量
7. 过敏体质者慎用
8. 本品性状发生改变时禁止使用
9. 如正在使用其他药品，使用前请咨询医师或药师

## 合规红线

1. **不得使用绝对化用语**："第一""唯一""最好""最安全"等表述均违反广告法，不得在任何对外内容中使用
2. **不得超适应症宣传**：说明书适应症为中毒性肝损伤辅助改善和脂肪肝/肝炎患者的食欲不振及右上腹压迫感，不得宣传降糖、降脂等未经批准的功效
3. **不得弱化禁忌和风险**：大豆过敏禁用、12岁以下禁用、妊娠/哺乳期不推荐等信息不得省略或弱化
4. **不得混淆药品与保健品**：本品为药品（OTC），不得将其定位或暗示为保健品/营养补充剂
5. **不得将动物实验结论等同于临床疗效**：说明书药理作用部分的动物模型数据不能直接作为临床疗效承诺
6. **不得声称治愈/根治**：本品定位为"辅助改善"，不得使用"治愈""根治""彻底消除"等表述
