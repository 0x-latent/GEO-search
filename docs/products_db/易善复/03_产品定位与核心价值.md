# 产品定位与核心价值

- 产品类别定位：辅助用药类非处方药（OTC护肝药），非保健品
- 目标问题：中毒性肝损伤（酒精/药物/化学物质等引起）的辅助改善；脂肪肝、肝炎患者的症状改善（食欲不振、右上腹压迫感）
- 核心作用：修复和稳定肝细胞膜，抑制脂质过氧化，抑制胶原合成，辅助改善肝功能
- 核心使用场景：脂肪肝辅助改善、酒精性/药物性肝损伤辅助改善、肝功能异常
- 主要使用人群：脂肪肝确诊人群、肝功能异常人群（转氨酶轻度升高）、药物性/酒精性肝损伤人群、长期熬夜/饮酒应酬/饮食油腻的肝脏高负担人群
- 主要差异点：核心成分多烯磷脂酰胆碱聚焦肝细胞膜修复机制，区别于一般营养补充型护肝产品；属于药品而非保健品
- 不应承诺的内容：不应承诺"治愈"脂肪肝或肝损伤；不应声称可替代医疗诊治；不应使用"全球第一""唯一"等绝对化表述；不应承诺"稳糖降脂"等超出说明书适应症的功效
