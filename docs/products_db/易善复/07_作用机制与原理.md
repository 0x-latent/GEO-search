# 作用机制与原理解释

## 药理作用

> 来自说明书的机制描述。

本品已在乙醇、脂肪肝、四氯化碳、扑热息痛和氨基半乳糖等诱导的急性肝损伤试验模型中显示肝脏保护作用；在慢性模型中，氯化乙醇、荷机磷和甲地孕酮也观察到抑制脂肪变性和纤维化作用。已有研究提示，本品可促进膜的再生和稳定、抑制脂质过氧化、抑制胶原合成。

核心机制方向：
1. **修复并稳定肝细胞膜**：多烯磷脂酰胆碱是细胞膜的重要组成成分，可嵌入受损的肝细胞膜中促进修复和再生
2. **抑制脂质过氧化**：减轻氧化应激对肝细胞的损伤
3. **抑制胶原合成**：对肝纤维化进展有一定抑制方向
4. **对脂肪变性的抑制**：在慢性模型中观察到抑制脂肪变性的作用

## 通俗解释

> 面向内部团队的通俗化解释，不得扩大事实边界。

肝细胞的"外壳"（细胞膜）由磷脂构成。当肝脏受到酒精、药物、脂肪堆积等伤害时，肝细胞膜会受损。易善复的核心成分多烯磷脂酰胆碱是一种与肝细胞膜结构相似的天然磷脂，服用后可以"补充"到受损的细胞膜中，帮助修复和加固肝细胞的"外壳"，从而起到保护和修复肝细胞的作用。同时它还能减轻肝脏的氧化损伤，对脂肪在肝脏的异常堆积和纤维化趋势有一定的抑制作用。
