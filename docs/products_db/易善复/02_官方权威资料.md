# 官方/权威资料核心信息

## 官方标准表达

> 保留原文或接近原文。

1. 本品含多烯磷脂酰胆碱，辅助改善中毒性肝损伤（如药物、毒物、化学物质和酒精引起的肝损伤）以及脂肪肝和肝炎患者的食欲不振和右上腹压迫感。（说明书适应症原文）
2. 本品为辅助治疗药，第一次使用前应咨询医师，治疗期间应定期到医院检查。（说明书注意事项）
3. 已在乙醇、脂肪肝、四氯化碳、扑热息痛和氨基半乳糖等诱导的急性肝损伤试验模型中显示肝脏保护作用。（说明书药理作用）
4. 在慢性模型中，氯化乙醇、荷机磷和甲地孕酮也观察到抑制脂肪变性和纤维化作用。（说明书药理作用）
5. 已有研究提示，本品可促进膜的再生和稳定、抑制脂质过氧化、抑制胶原合成。（说明书药理作用）
