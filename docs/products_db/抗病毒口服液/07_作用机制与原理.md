# 作用机制与原理解释

## 药理作用

> 来自说明书或权威资料的机制描述。

- 功能主治：清热祛湿，凉血解毒。用于风热感冒、流感。（来源：说明书）
- 组方功能概括：清热解毒、凉血利咽、清热泻火、化湿和中、芳香化浊、消肿散结。（来源：产品介绍资料）

### 主要成分与作用对应

| 成分 | 主要作用 | 对应症状 |
|---|---|---|
| 板蓝根、连翘 | 清热解毒 | 咽喉肿痛、风热感冒 |
| 石膏、知母 | 清热泻火、生津润燥 | 发热、热象明显 |
| 广藿香、石菖蒲 | 化湿和胃、芳香化浊 | 外感夹湿、胃口差、恶心 |
| 芦根 | 清热生津 | 发热口渴 |
| 地黄 | 凉血滋阴 | 热入血分、阴液耗伤 |
| 郁金 | 活血化瘀、清心凉血 | 热毒瘀滞 |

## 通俗解释

> 面向内部团队的通俗化解释，不得扩大事实边界。

本品是一个兼顾"清热解毒"和"化湿"的复方中药口服液。可以简单理解为：用板蓝根、连翘等"解毒"成分对付感冒热毒，用石膏、知母"退热降火"，同时用广藿香、石菖蒲"化湿和胃"——所以它不只是单纯的"抗病毒"概念，更适合处理风热、热毒、湿邪并见的感冒症状。
