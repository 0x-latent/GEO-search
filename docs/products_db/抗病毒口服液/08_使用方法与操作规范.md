# 使用方法与操作规范

## 用法用量

- 使用方式：口服
- 推荐剂量：一次10毫升（1支）
- 使用频次：一日2~3次（早饭前和午、晚饭后各服一次）
- 特殊人群：小儿酌减（具体用量未明确，建议咨询医生）

## 使用注意事项

- 忌烟、酒及辛辣、生冷、油腻食物
- 产品资料建议"尽早用药，全程用药"，适合在感冒/流感初期开始使用
- 其余注意事项详见说明书

## 不良反应

监测数据中可见以下不良反应：
- 胃肠道反应：恶心、呕吐、腹泻、腹痛、腹胀、腹部不适
- 皮肤反应：皮疹、瘙痒
- 其他：过敏反应等
