# 不适用/禁用/需谨慎场景

> 分为禁用场景、不建议自行使用场景、需谨慎场景三类。

## 禁用场景

### 场景：孕妇使用
- **场景类型**：禁用
- **原因**：说明书明确标注"孕妇禁用"。
- **标准处理建议**：明确告知孕妇不能使用本品，建议咨询医生选择孕期安全的感冒用药。
- **禁止表达**：可放心使用 / 少量没问题 / 怀孕早期可以用

### 场景：哺乳期妇女使用
- **场景类型**：禁用
- **原因**：说明书明确标注"哺乳期妇女禁用"。
- **标准处理建议**：明确告知哺乳期妇女不能使用本品，建议咨询医生。
- **禁止表达**：可放心使用 / 不影响喂奶 / 问题不大

## 不建议自行使用场景

### 场景：风寒感冒初期
- **场景类型**：不建议自行使用
- **原因**：本品以清热祛湿、凉血解毒为功效，偏寒凉，风寒感冒初期症状为畏寒重、清涕、无咽痛，证型不对症，使用可能加重寒象。
- **标准处理建议**：告知用户本品更适合风热感冒，风寒感冒建议选择辛温解表类产品或咨询药师。

### 场景：症状严重或持续不退
- **场景类型**：不建议自行使用
- **原因**：持续高热、呼吸困难、症状超过3天不退等情况可能提示重症或细菌感染，不应仅依赖OTC中成药。
- **标准处理建议**：建议及时就医，由医生评估是否需要抗病毒西药或抗生素治疗。

## 需谨慎场景

### 场景：胃肠道敏感人群
- **场景类型**：需谨慎
- **原因**：本品不良反应中包含恶心、呕吐、腹泻、腹痛、腹胀、腹部不适，胃肠道敏感者使用风险较高。
- **标准处理建议**：提醒用户注意观察胃肠道反应，如出现明显不适应停药并咨询医生。

### 场景：过敏体质人群
- **场景类型**：需谨慎
- **原因**：不良反应中包含皮疹、瘙痒、过敏反应等。
- **标准处理建议**：提醒过敏体质者使用前咨询医生或药师，用药后如出现皮疹、瘙痒等应立即停药就医。

### 场景：小儿用药
- **场景类型**：需谨慎
- **原因**：说明书仅提到"小儿酌减"，未给出明确的儿童用量标准。
- **标准处理建议**：建议家长咨询医生或药师确定具体用量，不建议自行估量。
