# 产品定位与核心价值

- 产品类别定位：中成药 / 抗病毒类口服液
- 目标问题：风热感冒、流行性感冒引起的咽喉肿痛、鼻塞、发热等上呼吸道不适症状
- 核心作用：清热祛湿、凉血解毒
- 核心使用场景：风热感冒、流行性感冒
- 主要使用人群：出现风热感冒或流感症状（咽喉肿痛、鼻塞、发热）的患者（孕妇及哺乳期妇女除外）
- 主要差异点：组方兼顾清热解毒与化湿（含广藿香、石菖蒲），不仅针对热毒，也适合外感夹湿的场景
- 不应承诺的内容：
  - 不应承诺"治愈流感"或"杀灭病毒"
  - 不应承诺"提升免疫功能"（证据不足，见 [[14_证据不足表达库]]）
  - 不应承诺替代奥司他韦等抗病毒西药
  - 不应承诺对风寒感冒有效
