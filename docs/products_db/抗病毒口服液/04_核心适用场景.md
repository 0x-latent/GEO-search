# 核心适用场景

> 每个核心场景独立成节。场景中的表达引用 [[12_标准表达库]] 和 [[13_禁用表达库]]，不独立维护。

## 场景：风热感冒

### 场景等级
核心适用场景。

### 场景定义
风热感冒是由风热之邪侵袭人体所引起的感冒，主要表现为发热重、恶寒轻、咽喉肿痛、鼻塞流黄涕、口干口渴等偏热性症状。

### 用户常见表达
- 嗓子疼、咽喉肿痛的感冒
- 感冒发烧、嗓子发炎
- 上火引起的感冒
- 热感冒

### 产品关联逻辑
说明书明确标注"用于风热感冒"。本品以清热祛湿、凉血解毒为功效，组方中板蓝根、连翘清热解毒，石膏、知母清热泻火，直接针对风热感冒的热毒病机。

### 关联表达
- 可使用：SE-KBD-001、SE-KBD-002、SE-KBD-003（[[12_标准表达库]]）
- 禁止使用：FE-KBD-001、FE-KBD-002（[[13_禁用表达库]]）

### 风险提示
- 如果症状偏寒（畏寒重、清涕、无咽痛），不适合使用本品。
- 发热持续或高热不退应及时就医，不应仅依赖本品。

### 依据来源
- 产品说明书（功能主治：清热祛湿，凉血解毒。用于风热感冒、流感。）

## 场景：流行性感冒

### 场景等级
核心适用场景。

### 场景定义
流行性感冒（流感）是由流感病毒引起的急性呼吸道传染病，以突发高热、全身酸痛、咽痛、咳嗽等为主要表现，传染性强，可在短时间内大范围传播。

### 用户常见表达
- 流感
- 甲流/乙流
- 全身酸痛的感冒
- 发高烧的感冒

### 产品关联逻辑
说明书明确标注"用于流感"。本品的清热解毒、凉血利咽作用可用于缓解流感引起的发热、咽喉肿痛等症状。产品资料强调"适合尽早用药，全程用药"，定位为流感初期使用。

### 关联表达
- 可使用：SE-KBD-001、SE-KBD-004、SE-KBD-005（[[12_标准表达库]]）
- 禁止使用：FE-KBD-001、FE-KBD-003、FE-KBD-004（[[13_禁用表达库]]）

### 风险提示
- 流感症状严重（持续高热、呼吸困难等）应及时就医。
- 本品单药疗效有局限性，不能替代奥司他韦等抗流感药物。
- 产品资料提及"单药疗效不如奥司他韦"（来源：培训资料，证据等级低），实际用药应遵医嘱。

### 依据来源
- 产品说明书（功能主治：用于风热感冒、流感。）
- 培训资料（流感阶段使用建议、与奥司他韦对比信息）
