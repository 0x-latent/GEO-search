# 合规与风险提示

## 禁忌

- 孕妇禁用（来源：说明书）
- 哺乳期妇女禁用（来源：说明书）

## 注意事项

- 忌烟、酒及辛辣、生冷、油腻食物（来源：说明书）
- 其余注意事项详见说明书

## 特殊人群提示

- 孕妇、哺乳期妇女：禁用
- 小儿：说明书仅注明"小儿酌减"，未给出具体用量，建议咨询医生
- 胃肠道敏感者：需注意恶心、呕吐、腹泻等不良反应风险
- 过敏体质者：需注意皮疹、瘙痒、过敏反应风险

## 合规红线

- 不得宣传"治愈感冒/流感"
- 不得宣传"杀灭病毒""消灭病毒"
- 不得宣传"提升免疫力/增强免疫功能"（证据不足，见 [[14_证据不足表达库]]）
- 不得宣传"辅助清除病毒"（证据不足，见 [[14_证据不足表达库]]）
- 不得将适用范围扩大到说明书以外的疾病
- 不得弱化孕妇、哺乳期妇女禁用的表述
- 不得在无证据情况下做与其他药品的优劣对比
- 不得使用绝对化用语（如"最好的""唯一的""完全安全"）
