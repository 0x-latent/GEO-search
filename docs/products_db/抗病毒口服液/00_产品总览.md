# 产品总览

> 本模块为摘要索引，不独立维护事实。具体内容请查阅对应模块。

- 产品类别：中成药（口服液）
- 产品核心功能：清热祛湿、凉血解毒
- 主要适用人群/场景：风热感冒、流行性感冒患者，表现为咽喉肿痛、鼻塞、发热等症状 → 详见 [[04_核心适用场景]]
- 核心价值：针对风热感冒和流感的中药复方口服液，清热解毒与祛湿并重 → 详见 [[03_产品定位与核心价值]]
- 主要使用方式：口服，一次10mL，一日2~3次 → 详见 [[08_使用方法与操作规范]]
- 主要风险/限制：孕妇、哺乳期妇女禁用；风寒感冒初期不对症 → 详见 [[06_不适用禁用谨慎场景]]
- 与同类产品的主要区别：含广藿香、石菖蒲等化湿成分，兼顾祛湿，不仅清热解毒 → 详见 [[09_产品差异与产品矩阵]]
- 资料来源：产品说明书（外盒）、产品介绍资料、培训资料
