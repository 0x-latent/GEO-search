# 产品总览

> 本模块为摘要索引，不独立维护事实。具体内容请查阅对应模块。

- 产品类别：复方营养补充类非处方药（OTC）
- 产品核心功能：补充钙、锌，辅助改善因缺钙缺锌引起的生长发育迟缓、食欲缺乏、骨质疏松等症状
- 主要适用人群/场景：缺钙缺锌引起的生长发育迟缓、食欲缺乏/厌食、骨质疏松、佝偻病等 → 详见 [[04_核心适用场景]]
- 核心价值：钙锌同补 + 赖氨酸辅助吸收，液体剂型方便儿童服用 → 详见 [[03_产品定位与核心价值]]
- 主要使用方式：口服，婴幼儿一日5-10ml，成人一日20-30ml，分2-3次饭后服 → 详见 [[08_使用方法与操作规范]]
- 主要风险/限制：血钙血锌过高者及甲状旁腺功能亢进者禁用；肾功能不全或糖尿病患者慎用 → 详见 [[06_不适用禁用谨慎场景]]
- 与同类产品的主要区别：复方制剂（钙+锌+赖氨酸），液体离子剂型，区别于单一补钙或单一补锌产品 → 详见 [[09_产品差异与产品矩阵]]
- 资料来源：药品说明书（2025年05月08日修订版）、产品培训资料
