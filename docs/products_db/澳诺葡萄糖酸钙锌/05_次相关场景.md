# 次相关场景

> 次相关场景的主要价值是"建立产品关联"，不是"强推荐"。

## 场景：日常营养补充（非疾病状态）

### 场景等级
次相关场景。

### 与产品的关系
产品可用于日常补充钙锌，适合生长发育期儿童、饮食不均衡人群等的日常营养补充需求。

### 可提及逻辑
当用户询问日常补钙补锌需求、饮食不均衡时的营养补充时，可提及本品作为钙锌补充选择之一。

### 不应直接推荐的原因
日常营养补充的首选方案应为均衡膳食，非疾病状态下不应过度推荐药物类产品；且无明确缺钙缺锌依据时，补充的必要性因人而异。

### 关联表达
- 可使用：SE-AN-001、SE-AN-009 → [[12_标准表达库]]
- 禁止使用：FE-AN-001、FE-AN-003、FE-AN-005 → [[13_禁用表达库]]

### 风险提示
- 不能将日常补充等同于治疗需求。
- 应建议用户先评估饮食结构，必要时通过微量元素检测明确是否需要补充。

## 场景：挑食偏食（非疾病状态）

### 场景等级
次相关场景。

### 与产品的关系
挑食偏食可能导致钙锌摄入不足，与产品的补充功能有一定关联。

### 可提及逻辑
当用户反映孩子挑食偏食、担心营养不足时，可提及本品作为辅助补充选择，但应优先建议改善饮食习惯。

### 不应直接推荐的原因
挑食偏食本身不是疾病，原因涉及饮食习惯、心理行为等多方面，不应简单以药物补充解决；且未经检测不确定是否真正缺钙缺锌。

### 关联表达
- 可使用：SE-AN-001、SE-AN-009 → [[12_标准表达库]]
- 禁止使用：FE-AN-001、FE-AN-003、FE-AN-005 → [[13_禁用表达库]]

### 风险提示
- 不能将"挑食"直接等同于"缺钙缺锌"。
- 应建议家长关注饮食结构调整，必要时就医评估。
