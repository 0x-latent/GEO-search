# 使用方法与操作规范

## 用法用量

- 使用方式：口服，饭后服
- 推荐剂量：
  - 婴幼儿：一日5-10ml
  - 成人：一日20-30ml
- 使用频次：分2-3次服用
- 疗程建议（来自产品资料）：30天为一个疗程，3个疗程为一个周期

## 使用注意事项

1. 饭后服用，减少胃肠刺激。
2. 儿童必须在成人监护下使用。
3. 本品性状发生改变时禁止使用。
4. 请将本品放在儿童不能接触的地方。
5. 如正在使用其他药品，使用本品前请咨询医师或药师。

## 药物相互作用

1. 不宜与洋地黄类药物合用（钙可增强洋地黄毒性）。
2. 不要与四环素、青霉胺、苯妥英钠以及磷酸盐同用。
3. 如与其他药物同时使用可能会发生药物相互作用，详情请咨询医师或药师。

## 不良反应

1. 可见轻度恶心、呕吐、便秘等反应。
2. 长期服用可引起反跳性胃酸分泌增高。
