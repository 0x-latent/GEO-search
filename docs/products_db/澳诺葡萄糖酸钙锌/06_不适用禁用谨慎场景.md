# 不适用/禁用/需谨慎场景

> 分为禁用场景、不建议自行使用场景、需谨慎场景三类。

## 禁用场景

### 场景：血钙或血锌过高者
- **场景类型**：禁用
- **原因**：说明书明确列为禁忌。血钙或血锌已偏高的情况下继续补充可导致高钙血症或高锌血症，引发严重不良反应。
- **标准处理建议**：明确告知不可使用，建议就医检查后遵医嘱。
- **禁止表达**：可放心使用 / 没问题 / 少量应该没事

### 场景：甲状旁腺功能亢进患者
- **场景类型**：禁用
- **原因**：说明书明确列为禁忌。甲状旁腺功能亢进本身可导致高钙血症，额外补钙会加重病情。
- **标准处理建议**：明确告知不可使用，建议在内分泌科医生指导下处理钙代谢问题。
- **禁止表达**：可放心使用 / 没问题 / 直接用即可

### 场景：对本品过敏者
- **场景类型**：禁用
- **原因**：说明书注意事项明确"对本品过敏者禁用"。
- **标准处理建议**：了解过敏史，若有过敏反应需停药。
- **禁止表达**：可放心使用 / 没问题

## 不建议自行使用场景

### 场景：不明原因的严重生长发育落后
- **场景类型**：不建议自行使用
- **原因**：严重的生长发育落后可能涉及内分泌、遗传等复杂因素，单纯补钙锌可能延误诊治。
- **标准处理建议**：建议先到儿童保健科或内分泌科就诊，明确原因后再决定是否需要补充钙锌。

### 场景：严重或持续性手足抽搐
- **场景类型**：不建议自行使用
- **原因**：严重抽搐可能需要紧急静脉补钙等急性处理，口服补钙不能替代急性治疗。
- **标准处理建议**：建议立即就医，由医生判断病因和处理方案。

## 需谨慎场景

### 场景：肾功能不全患者
- **场景类型**：需谨慎
- **原因**：说明书注意事项明确"肾功能不全患者慎用"。肾功能不全可能影响钙的排泄，增加高钙血症风险。
- **标准处理建议**：建议在医生指导下使用，定期监测血钙水平。

### 场景：糖尿病患者
- **场景类型**：需谨慎
- **原因**：说明书注意事项明确"糖尿病患者慎用"。产品含葡萄糖酸盐成分。
- **标准处理建议**：建议在医生指导下使用，关注血糖变化。

### 场景：过敏体质者
- **场景类型**：需谨慎
- **原因**：说明书注意事项明确"过敏体质者慎用"。
- **标准处理建议**：建议首次使用时注意观察是否出现过敏反应，如有不适及时停药就医。

### 场景：正在使用洋地黄类药物的患者
- **场景类型**：需谨慎
- **原因**：说明书药物相互作用明确"不宜与洋地黄类药物合用"。钙可增强洋地黄的毒性。
- **标准处理建议**：告知不宜合用，建议咨询医生。

### 场景：正在使用四环素、青霉胺、苯妥英钠或磷酸盐的患者
- **场景类型**：需谨慎
- **原因**：说明书药物相互作用明确"不要与四环素、青霉胺、苯妥英钠以及磷酸盐同用"。
- **标准处理建议**：告知不宜合用，建议咨询医师或药师调整用药方案。
