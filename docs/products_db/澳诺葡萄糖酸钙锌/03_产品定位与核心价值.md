# 产品定位与核心价值

- 产品类别定位：钙锌同补的复方营养性药品（OTC）
- 目标问题：因缺钙、缺锌引起的生长发育迟缓、食欲缺乏、骨骼问题等
- 核心作用：同时补充钙和锌，并通过赖氨酸辅助促进钙锌吸收
- 核心使用场景：儿童生长发育期钙锌补充、缺钙缺锌相关症状的辅助治疗、孕产期及绝经期妇女钙补充
- 主要使用人群：生长发育期儿童及婴幼儿、有缺钙缺锌症状的各年龄段人群、孕产期及绝经期妇女
- 主要差异点：钙锌同补（非单一成分）、液体口服溶液剂型（便于儿童服用）、含赖氨酸辅助吸收
- 不应承诺的内容：不应承诺"一定能长高""一定能改善食欲"等确定性疗效；不应承诺可替代正规医疗诊治；不应宣称适合所有人群（禁忌人群明确）
