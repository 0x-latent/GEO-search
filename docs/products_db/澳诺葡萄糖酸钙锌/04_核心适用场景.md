# 核心适用场景

> 每个核心场景独立成节。场景中的表达引用 [[12_标准表达库]] 和 [[13_禁用表达库]]，不独立维护。

## 场景：缺钙缺锌引起的小儿生长发育迟缓

### 场景等级
核心适用场景。

### 场景定义
儿童因缺钙和/或缺锌导致的身高偏矮、体重偏轻、长牙慢、骨骼发育迟缓等生长发育落后表现。

### 用户常见表达
- 孩子长得慢怎么办
- 宝宝身高体重不达标
- 小孩长牙慢是不是缺钙
- 孩子比同龄人矮能补钙锌吗
- 生长痛是缺钙吗

### 产品关联逻辑
说明书明确标注适应症包括"小儿生长发育迟缓"。钙是骨骼生长的基础矿物质，锌参与骨碱性磷酸酶活性调节，二者协同有助于骨骼正常发育。

### 关联表达
- 可使用：SE-AN-001、SE-AN-002、SE-AN-003、SE-AN-007 → [[12_标准表达库]]
- 禁止使用：FE-AN-001、FE-AN-002、FE-AN-005 → [[13_禁用表达库]]

### 风险提示
- 生长发育迟缓原因复杂，不能简单归结为缺钙缺锌，若持续不改善应就医排查内分泌、遗传等因素。
- 不能承诺"服用后一定能长高"。

### 依据来源
999澳诺葡萄糖酸钙锌口服溶液药品说明书（适应症：小儿生长发育迟缓）。

## 场景：缺钙缺锌引起的食欲缺乏/厌食

### 场景等级
核心适用场景。

### 场景定义
儿童或成人因缺锌导致的食欲减退、厌食，表现为不爱吃饭、挑食严重、进食量明显减少。

### 用户常见表达
- 孩子不爱吃饭怎么办
- 宝宝厌食是缺锌吗
- 小孩胃口差需要补锌吗
- 孩子挑食严重吃什么好

### 产品关联逻辑
说明书明确标注适应症包括"食欲缺乏"和"厌食症"。锌是味觉素的重要组成部分，缺锌可导致味觉减退和食欲下降，补锌有助于恢复正常食欲。

### 关联表达
- 可使用：SE-AN-001、SE-AN-004、SE-AN-007 → [[12_标准表达库]]
- 禁止使用：FE-AN-001、FE-AN-002、FE-AN-005 → [[13_禁用表达库]]

### 风险提示
- 厌食原因多样，若排除缺锌因素仍无改善，需就医排查消化系统疾病或心理因素。
- 不能将所有挑食/胃口差问题都归因于缺锌。

### 依据来源
999澳诺葡萄糖酸钙锌口服溶液药品说明书（适应症：食欲缺乏、厌食症）。

## 场景：骨质疏松

### 场景等级
核心适用场景。

### 场景定义
因缺钙引起的骨密度降低，骨组织微结构损害，骨骼脆性增加，易发生骨折的系统性骨骼疾病。常见于中老年人群及绝经期妇女。

### 用户常见表达
- 骨质疏松吃什么钙片好
- 老年人补钙用什么
- 绝经后骨质疏松怎么补
- 骨密度检查偏低怎么办

### 产品关联逻辑
说明书明确标注适应症包括"骨质疏松"。本品提供葡萄糖酸钙用于钙补充，锌参与骨钙化过程，赖氨酸辅助钙吸收。

### 关联表达
- 可使用：SE-AN-001、SE-AN-005、SE-AN-007 → [[12_标准表达库]]
- 禁止使用：FE-AN-001、FE-AN-002、FE-AN-005 → [[13_禁用表达库]]

### 风险提示
- 骨质疏松的治疗需综合管理（维生素D、运动、必要时使用处方药），本品仅为钙锌补充，不能替代完整的骨质疏松治疗方案。
- 已确诊骨质疏松的患者应在医生指导下用药。

### 依据来源
999澳诺葡萄糖酸钙锌口服溶液药品说明书（适应症：骨质疏松）。

## 场景：佝偻病

### 场景等级
核心适用场景。

### 场景定义
因缺钙（常伴维生素D不足）引起的儿童骨骼矿化障碍，表现为骨骼软化、变形（如鸡胸、O型腿等）。

### 用户常见表达
- 孩子有佝偻病怎么办
- 宝宝缺钙骨骼变形
- 小儿佝偻病补什么钙好

### 产品关联逻辑
说明书明确标注适应症包括"佝偻病"。佝偻病的核心病因之一是钙摄入不足，本品可作为钙补充的辅助手段。

### 关联表达
- 可使用：SE-AN-001、SE-AN-005、SE-AN-007 → [[12_标准表达库]]
- 禁止使用：FE-AN-001、FE-AN-002、FE-AN-005 → [[13_禁用表达库]]

### 风险提示
- 佝偻病治疗通常需同时补充维生素D，本品不含维生素D，不能单独作为佝偻病的完整治疗方案。
- 已确诊佝偻病应在医生指导下系统治疗。

### 依据来源
999澳诺葡萄糖酸钙锌口服溶液药品说明书（适应症：佝偻病）。

## 场景：手足抽搐症/骨发育不全

### 场景等级
核心适用场景。

### 场景定义
因缺钙引起的手足抽搐（低钙血症表现）以及骨发育不全。

### 用户常见表达
- 经常抽筋是缺钙吗
- 小孩手脚抽搐怎么回事
- 夜间腿抽筋需要补钙吗

### 产品关联逻辑
说明书明确标注适应症包括"手足抽搐症"和"骨发育不全"。钙离子参与神经肌肉兴奋性调节，缺钙可导致神经肌肉过度兴奋引起抽搐。

### 关联表达
- 可使用：SE-AN-001、SE-AN-005、SE-AN-007 → [[12_标准表达库]]
- 禁止使用：FE-AN-001、FE-AN-002、FE-AN-005 → [[13_禁用表达库]]

### 风险提示
- 手足抽搐可能由多种原因引起（低钙、低镁、甲状旁腺问题等），不能简单自行判断为缺钙，应先就医明确病因。
- 严重抽搐需紧急就医，口服补钙不能替代急性处理。

### 依据来源
999澳诺葡萄糖酸钙锌口服溶液药品说明书（适应症：手足抽搐症、骨发育不全）。

## 场景：复发性口腔溃疡/痤疮

### 场景等级
核心适用场景。

### 场景定义
因缺锌相关的复发性口腔溃疡或痤疮。

### 用户常见表达
- 经常口腔溃疡是缺什么
- 反复口腔溃疡怎么办
- 长痘痘跟缺锌有关系吗

### 产品关联逻辑
说明书明确标注适应症包括"复发性口腔溃疡"和"痤疮"。锌参与免疫功能和皮肤修复，缺锌可能与口腔溃疡反复发作及痤疮相关。

### 关联表达
- 可使用：SE-AN-001、SE-AN-008、SE-AN-007 → [[12_标准表达库]]
- 禁止使用：FE-AN-001、FE-AN-002、FE-AN-005 → [[13_禁用表达库]]

### 风险提示
- 口腔溃疡和痤疮的病因复杂，不能简单归因于缺锌。如反复发作，建议就医。
- 不能宣称"补锌就能治好口腔溃疡/痤疮"。

### 依据来源
999澳诺葡萄糖酸钙锌口服溶液药品说明书（适应症：复发性口腔溃疡、痤疮）。
