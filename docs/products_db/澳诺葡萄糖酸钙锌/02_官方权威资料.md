# 官方/权威资料核心信息

## 官方标准表达

> 保留原文或接近原文。

- 功能主治：用于治疗因缺钙、缺锌引起的疾病，包括骨质疏松、手足抽搐症、骨发育不全、佝偻病、妊娠妇女和哺乳期妇女以及绝经期妇女的钙补充、小儿生长发育迟缓、食欲缺乏、厌食症、复发性口腔溃疡、痤疮等。
- 禁忌：血钙、血锌过高者禁用；甲状旁腺功能亢进患者禁用。
- 不良反应：可见轻度恶心、呕吐、便秘等反应；长期服用可引起反跳性胃酸分泌增高。
- 用法用量：婴幼儿一日5-10ml，成人一日20-30ml，分2-3次，饭后服。
