# 合规与风险提示

## 禁忌

- 血钙、血锌过高者禁用。
- 甲状旁腺功能亢进患者禁用。
- 对本品过敏者禁用。

## 注意事项

1. 肾功能不全或糖尿病患者慎用。
2. 过敏体质者慎用。
3. 本品性状发生改变时禁止使用。
4. 请将本品放在儿童不能接触的地方。
5. 儿童必须在成人监护下使用。
6. 如正在使用其他药品，使用本品前请咨询医师或药师。

## 特殊人群提示

- **婴幼儿**：需在成人监护下使用，按说明书推荐剂量（一日5-10ml）服用。
- **孕产妇**：说明书适应症包括妊娠妇女和哺乳期妇女的钙补充，但用药应在医生指导下进行。
- **老年人**：骨质疏松患者可使用，但应注意排除禁忌症（如甲状旁腺功能亢进、肾功能不全）。
- **肾功能不全患者**：慎用，需监测血钙水平。
- **糖尿病患者**：慎用，注意血糖变化。

## 合规红线

1. 不得宣称"治愈""根治"任何疾病。
2. 不得隐瞒禁忌人群和不良反应信息。
3. 不得将适应症范围扩大到说明书以外。
4. 不得做无依据的竞品贬低。
5. 不得替代医学诊断（如"你的孩子就是缺钙"）。
6. "高解离率""高吸收率""高利用率""配比科学""更容易吸收"等表述缺少临床证据，不得作为对外宣传话术。
7. 不得将"禁用"人群弱化表述为"慎用"。
