# 次相关场景

> 次相关场景的主要价值是"建立产品关联"，不是"强推荐"。以下场景来源于培训资料扩展，非说明书原文，证据等级为C级（企业培训资料）。

## 场景：反酸

### 场景等级
次相关场景。

### 与产品的关系
反酸是慢性胃炎的常见伴随症状之一。养胃舒组方中乌梅具有收敛作用，培训资料将反酸列入养胃舒的常见适用症状。但说明书功能主治未直接提及"反酸"。

### 可提及逻辑
当用户因慢性胃炎伴反酸症状咨询时，可说明养胃舒适用于慢性胃炎，反酸作为慢性胃炎的常见伴随症状可能在调理过程中得到缓解。

### 不应直接推荐的原因
- 说明书功能主治未直接包含"反酸"。
- 反酸的病因多样（胃食管反流、消化性溃疡等），需要明确诊断。

### 关联表达
- 可使用：SE-YWS-006 → [[12_标准表达库]]
- 禁止使用：FE-YWS-004 → [[13_禁用表达库]]

### 风险提示
- 频繁或严重反酸应就医排查病因，不应自行长期用药。
- 不可宣称养胃舒"治疗反酸"或"抑制胃酸"。

## 场景：胃胀、餐后饱胀

### 场景等级
次相关场景。

### 与产品的关系
胃胀、餐后饱胀是慢性胃炎的常见伴随症状。养胃舒组方中陈皮理气健脾、山楂消食，培训资料将胃胀列入适用范围。但说明书功能主治未直接提及。

### 可提及逻辑
当用户因慢性胃炎伴胃胀、餐后饱胀咨询时，可说明养胃舒适用于慢性胃炎，胃胀作为慢性胃炎的常见伴随表现可能在调理过程中有所改善。

### 不应直接推荐的原因
- 说明书功能主治未直接包含"胃胀"或"餐后饱胀"。
- 胃胀的病因多样，需排除功能性消化不良、胃动力障碍等。

### 关联表达
- 可使用：SE-YWS-007 → [[12_标准表达库]]
- 禁止使用：FE-YWS-004 → [[13_禁用表达库]]

### 风险提示
- 持续性胃胀或伴有其他严重症状应就医。
- 不可宣称养胃舒"治疗胃胀"。

## 场景：恶心、消化不良

### 场景等级
次相关场景。

### 与产品的关系
恶心和消化不良是慢性胃炎的常见伴随症状。培训资料将其列入养胃舒的常见适用症状，组方中山楂消食、陈皮理气有助于消化功能。但说明书功能主治未直接提及。

### 可提及逻辑
当用户因慢性胃炎伴恶心、消化不良咨询时，可说明养胃舒适用于慢性胃炎，这些症状作为伴随表现可能在调理中有所缓解。

### 不应直接推荐的原因
- 说明书功能主治未直接包含"恶心"或"消化不良"。
- 恶心和消化不良的病因复杂，需排除其他疾病。

### 关联表达
- 可使用：SE-YWS-008 → [[12_标准表达库]]
- 禁止使用：FE-YWS-004 → [[13_禁用表达库]]

### 风险提示
- 持续或严重恶心（尤其伴呕吐、体重下降）应就医。
- 不可宣称养胃舒"治疗消化不良"或"止吐"。

## 场景：饮食刺激后胃不适

### 场景等级
次相关场景。

### 与产品的关系
因饮食油腻、辛辣、酒精、冰饮、咖啡等刺激导致的胃部不适，培训资料将其列入养胃舒的适用人群。养胃舒以滋阴养胃为主，对刺激后的胃黏膜养护有理论关联。

### 可提及逻辑
当用户因饮食刺激后出现胃部不适咨询时，如伴有灼热、隐痛等养胃舒核心场景的症状，可建议参考。

### 不应直接推荐的原因
- 说明书功能主治针对"慢性胃炎、胃脘灼热、隐隐作痛"，非针对饮食刺激后的急性不适。
- 饮食刺激后的不适通常为一过性，不一定需要用药。

### 关联表达
- 可使用：SE-YWS-009 → [[12_标准表达库]]
- 禁止使用：FE-YWS-004 → [[13_禁用表达库]]

### 风险提示
- 如因饮食刺激出现剧烈腹痛、呕血等急症表现，应立即就医。
- 不可将一次性饮食不当等同于慢性胃炎。
