# 作用机制与原理解释

## 药理作用

> 来自说明书或权威资料的机制描述。

说明书功能主治为"滋阴养胃"，未详细列出药理机制。以下为组方的中医理论解读：

- **健脾益气**：党参、白术(炒)、山药 — 健脾益气，补益脾胃之阴
- **养阴益胃**：北沙参、玄参、黄精(蒸) — 养阴生津，滋养胃阴
- **理气消食**：陈皮、山楂(炒) — 理气健脾、消食化积
- **温补脾阳**：干姜、菟丝子 — 温中散寒、温补脾肾
- **生津收敛**：乌梅 — 生津止渴、收敛固涩

整体组方特点为"阴阳并调、寒温并用"，以滋阴养胃为主，兼顾健脾理气。

## 通俗解释

> 面向内部团队的通俗化解释，不得扩大事实边界。

养胃舒颗粒以"滋阴养胃"为核心，通过多味中药协同作用，一方面滋养胃部阴液，缓解因胃阴不足导致的灼热、隐痛；另一方面健脾理气，促进消化功能。组方中阴阳兼顾、寒温并用，药性相对温和。
