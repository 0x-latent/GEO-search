# 产品定位与核心价值

- 产品类别定位：中成药（OTC），滋阴养胃类胃药
- 目标问题：慢性胃炎及其引起的胃脘灼热、隐隐作痛
- 核心作用：滋阴养胃
- 核心使用场景：慢性胃炎、胃脘灼热、隐隐作痛
- 主要使用人群：慢性胃炎患者、胃脘灼热隐痛人群、胃部反复不适需调理养护的人群
- 主要差异点：与三九胃泰（清热柔肝止痛、偏寒凉、中重度+短期治疗）相比，养胃舒偏滋阴养胃、药性温和、适合轻中度+长期调理
- 不应承诺的内容：
  - 不可承诺"治愈"慢性胃炎
  - 不可承诺适用于所有类型的胃病或胃痛
  - 不可承诺"无副作用"或"绝对安全"（不良反应和禁忌均尚不明确）
  - 不可承诺替代就医诊疗
