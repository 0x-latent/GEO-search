# 合规与风险提示

## 禁忌

- 说明书标注：尚不明确。
- 对本品过敏者禁用。
- 本品性状发生改变时禁止使用。

## 注意事项

1. 孕妇慎用。
2. 湿热胃痛证及重度胃痛应在医师指导下服用。
3. 糖尿病患者、儿童及年老体虚者应在医师指导下服用。
4. 服本药三天症状未改善，应停止服用，并去医院就诊。
5. 对本品过敏者禁用，过敏体质者慎用。
6. 本品性状发生改变时禁止使用。
7. 儿童必须在成人监护下使用。
8. 请将本品放在儿童不能接触的地方。
9. 如正在使用其他药品，使用本品前请咨询医师或药师。

## 特殊人群提示

- **孕妇**：慎用，需咨询医师。
- **儿童**：须在医师指导下服用，必须由成人监护。
- **年老体虚者**：须在医师指导下服用。
- **糖尿病患者**：须在医师指导下服用（辅料含蔗糖）。
- **过敏体质者**：慎用。

## 合规红线

- **不可宣称治愈**：慢性胃炎为慢性疾病，不可承诺"治愈""根治"。
- **不可扩大适应症**：功能主治仅限"慢性胃炎、胃脘灼热、隐隐作痛"，不可扩展至所有胃病或胃痛。
- **不可宣称无副作用**："尚不明确"不等于"无不良反应"，不可用"绝对安全""零副作用"等表述。
- **不可省略就医提示**：湿热胃痛、重度胃痛需医师指导，3天无改善需就医，这些限制不可省略。
- **不可做无依据的竞品比较**：与三九胃泰等产品仅可做客观差异说明，不可做优劣判断。
- **营销表达需审查**：培训资料中的"疗效确切""多靶点修复""药食同源""可长期服用"等表述证据不足，不可直接对外使用。
