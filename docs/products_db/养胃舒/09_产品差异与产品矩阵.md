# 产品差异与产品矩阵

> 仅基于客观维度对比（成分、剂型、适用范围、使用方式、人群限制等），不做无依据的优劣判断。

## 三九胃泰家族产品矩阵

培训资料中对家族产品分工的总结：
- **脾胃虚寒** → 温胃舒
- **多年老胃病**（经常胃痛、胃酸反流、恶心呕吐等中重度症状） → 三九胃泰
- **偶发胃痛胃不适、慢性胃炎、养胃需求** → 养胃舒

> 来源：企业培训资料，证据等级C。

## 养胃舒 vs 三九胃泰 详细对比

| 对比维度 | 养胃舒颗粒 | 三九胃泰颗粒 |
|---|---|---|
| 核心成分 | 党参、陈皮、黄精(蒸)、山药、玄参、乌梅、山楂(炒)、北沙参、干姜、菟丝子、白术(炒) | 三叉苦、九里香、两面针、木香、黄芩、茯苓、地黄、白芍（另有颗粒剂含对乙酰氨基酚等西药成分） |
| 核心作用 | 滋阴养胃、止痛消炎、养护 | 清热、柔肝、活血止痛 |
| 药性特点 | 药性温和，适用面较广 | 药性偏寒凉，辨证更明确 |
| 剂型/形态 | 颗粒剂（低糖型） | 颗粒剂 |
| 适用范围 | 慢性胃炎、胃脘灼热、隐隐作痛；适合轻中度 + 长期调理 | 浅表性胃炎、糜烂性胃炎、萎缩性胃炎等；适合中重度 + 短期治疗 |
| 适用场景 | 偶发胃痛、慢性胃炎、胃部亚健康、日常养胃 | 经常胃痛、胃酸反流、恶心呕吐、胃热或明显炎症刺激 |
| 使用方式 | 开水冲服，一次1～2袋，一日2次 | 开水冲服（具体用量见说明书） |
| 人群限制 | 孕妇慎用；湿热胃痛及重度胃痛需医师指导 | 见三九胃泰说明书 |
| 证据来源 | 药品说明书 + 企业培训资料 | 药品说明书 + 企业培训资料 |

## 场景化选品参考

| 用户场景 | 推荐优先级 | 说明 |
|---|---|---|
| 偶发胃痛、轻度不适 | 养胃舒优先 | 药性温和，适合轻中度 |
| 慢性胃炎长期调理 | 养胃舒优先 | 说明书直接对应，适合长期使用 |
| 胃火重、症状明显、发作急 | 三九胃泰优先 | 清热止痛，对症更直接 |
| 胃酸反流严重、恶心呕吐 | 三九胃泰优先 | 中重度症状更适合 |
| 脾胃虚寒（腹痛喜暖泄泻） | 温胃舒 | 养胃舒和三九胃泰均不是首选 |
| 日常养胃、胃部亚健康 | 养胃舒优先 | 培训资料定位为养胃调理 |

> 以上场景化选品参考来源为企业培训资料，证据等级C，仅供内部参考。
