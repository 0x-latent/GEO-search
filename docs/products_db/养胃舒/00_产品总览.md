# 产品总览

> 本模块为摘要索引，不独立维护事实。具体内容请查阅对应模块。

- 产品类别：中成药（OTC）
- 产品核心功能：滋阴养胃，用于慢性胃炎，胃脘灼热、隐隐作痛
- 主要适用人群/场景：慢性胃炎患者、胃脘灼热隐痛人群 → 详见 [[04_核心适用场景]]
- 核心价值：滋阴养胃，兼顾治疗与养护，药性温和适用面广 → 详见 [[03_产品定位与核心价值]]
- 主要使用方式：开水冲服，一次1～2袋，一日2次 → 详见 [[08_使用方法与操作规范]]
- 主要风险/限制：孕妇慎用；湿热胃痛及重度胃痛应在医师指导下服用；3天无改善应就医 → 详见 [[06_不适用禁用谨慎场景]]
- 与同类产品的主要区别：养胃舒偏滋阴养胃+长期调理，三九胃泰偏清热止痛+短期治疗 → 详见 [[09_产品差异与产品矩阵]]
- 资料来源：养胃舒颗粒药品说明书（批准文号Z34020289）、企业培训资料
