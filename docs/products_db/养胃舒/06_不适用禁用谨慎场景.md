# 不适用/禁用/需谨慎场景

> 分为禁用场景、不建议自行使用场景、需谨慎场景三类。

## 禁用场景

### 场景：对本品成分过敏
- **场景类型**：禁用
- **原因**：说明书明确标注"对本品过敏者禁用"。
- **标准处理建议**：明确告知禁用，建议更换其他药物。
- **禁止表达**：可放心使用 / 试试看 / 过敏不严重没关系

### 场景：药品性状发生改变
- **场景类型**：禁用
- **原因**：说明书明确标注"本品性状发生改变时禁止使用"。
- **标准处理建议**：告知不可使用，更换新药。
- **禁止表达**：应该没事 / 还能吃

## 不建议自行使用场景

### 场景：湿热胃痛证
- **场景类型**：不建议自行使用
- **原因**：说明书明确标注"湿热胃痛证应在医师指导下服用"。养胃舒以滋阴养胃为主，湿热胃痛的病机与之不同，不当使用可能无效甚至加重。
- **标准处理建议**：建议在医师指导下使用，不可自行决定。

### 场景：重度胃痛
- **场景类型**：不建议自行使用
- **原因**：说明书明确标注"重度胃痛应在医师指导下服用"。养胃舒适用于隐隐作痛等轻中度不适，重度胃痛需排除严重器质性病变。
- **标准处理建议**：建议就医明确诊断，在医师指导下用药。

### 场景：糖尿病患者
- **场景类型**：不建议自行使用
- **原因**：说明书标注"糖尿病患者应在医师指导下服用"，辅料含蔗糖。
- **标准处理建议**：建议在医师指导下使用。

### 场景：儿童用药
- **场景类型**：不建议自行使用
- **原因**：说明书标注"儿童应在医师指导下服用""儿童必须在成人监护下使用"。
- **标准处理建议**：建议在医师指导下使用，必须由成人监护。

### 场景：年老体虚者
- **场景类型**：不建议自行使用
- **原因**：说明书标注"年老体虚者应在医师指导下服用"。
- **标准处理建议**：建议在医师指导下使用。

## 需谨慎场景

### 场景：孕妇
- **场景类型**：需谨慎
- **原因**：说明书明确标注"孕妇慎用"。
- **标准处理建议**：慎用，建议咨询医师后决定是否使用。

### 场景：过敏体质者
- **场景类型**：需谨慎
- **原因**：说明书标注"过敏体质者慎用"。
- **标准处理建议**：慎用，使用中出现过敏反应应立即停药。

### 场景：3天症状无改善
- **场景类型**：需谨慎（实为要求停药就医）
- **原因**：说明书明确标注"服本药三天症状未改善，应停止服用，并去医院就诊"。
- **标准处理建议**：停药并就医，不可继续自行用药。

### 场景：正在使用其他药品
- **场景类型**：需谨慎
- **原因**：说明书标注"如正在使用其他药品，使用本品前请咨询医师或药师"。
- **标准处理建议**：建议咨询医师或药师后再决定是否联合使用。
