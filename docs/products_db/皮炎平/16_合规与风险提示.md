# 合规与风险提示

## 禁忌

### 绿皮
- 对本品任何成份过敏者禁用
- 不得用于皮肤破溃处
- 本品性状发生改变时禁止使用

### 红皮
- 患处已破溃、化脓或有明显渗出者禁用
- 病毒感染者（如有疱疹、水痘）禁用
- 对本品过敏者禁用
- 本品性状发生改变时禁止使用

## 注意事项

- 避免接触眼睛和其他黏膜（如口、鼻等）
- 不宜大面积、长期使用
- 用药7日（绿皮）/ 1周（红皮）后症状未缓解，应咨询医师或药师
- 用药部位如有烧灼感、红肿等情况应停药，并将局部药物洗净，必要时向医师咨询
- 如并发细菌或真菌感染，请咨询医师处理
- 如正在使用其他药品，使用本品前请咨询医师或药师
- 请将本品放在儿童不能接触的地方

## 特殊人群提示

| 人群 | 绿皮 | 红皮 |
|---|---|---|
| 孕妇及哺乳期妇女 | 慎用 | 慎用 |
| 婴幼儿 | 更敏感，使用应谨慎 | 小儿避免使用 |
| 儿童 | 更敏感，须成人监护下使用 | 小儿避免使用，须成人监护 |
| 皮肤萎缩的老年人 | 更敏感，使用应谨慎 | 谨慎使用 |
| 过敏体质者 | 慎用 | 慎用 |
| 运动员 | 慎用 | 慎用 |

## 合规红线

1. **不得扩大适应症**：对外内容中适应症表述不得超出说明书记载范围
2. **不得弱化风险**：不得使用"完全安全""无副作用""放心使用"等表述
3. **不得做价格承诺**：具体零售价格和医保报销比例不应在对外内容中固定化
4. **不得做竞品贬低**：不得无依据地声称"比XX更安全/更有效/副作用更低"
5. **不得给出明确诊疗方案**：联合用药方案不得以确定语气推荐，应建议咨询医师
6. **不得给出明确年龄界限**：说明书未标注具体年龄（如2岁），不得自行添加
7. **不得引用未经核实的权威来源**：如"FDA批准"等需有明确出处
8. **不得指导急救操作**：如催吐等专业操作不应在对外内容中指导
