# 不适用/禁用/需谨慎场景

> 分为禁用场景、不建议自行使用场景、需谨慎场景三类。

## 禁用场景

### 场景：皮肤破溃处
- **场景类型**：禁用
- **原因**：绿皮说明书明确"不得用于皮肤破溃处"；红皮说明书明确"患处已破溃、化脓或有明显渗出者禁用"
- **标准处理建议**：破损皮肤应先进行创面处理，待愈合后再考虑使用
- **禁止表达**：可放心使用 / 没问题 / 破了也能涂

### 场景：病毒感染（疱疹、水痘）
- **场景类型**：禁用（红皮）
- **原因**：红皮说明书明确"病毒感染者（如有疱疹、水痘）禁用"。外用糖皮质激素可能加重病毒感染
- **标准处理建议**：建议就医，使用抗病毒药物治疗
- **禁止表达**：可放心使用 / 涂一点没关系

### 场景：对成分过敏者
- **场景类型**：禁用
- **原因**：绿皮说明书"对本品任何成份过敏者禁用"；红皮说明书"对本品过敏者禁用"
- **标准处理建议**：停止使用，如已出现过敏反应应就医处理
- **禁止表达**：用量少就不过敏 / 试试看

### 场景：真菌感染（足癣、体癣等）
- **场景类型**：禁用
- **原因**：外用糖皮质激素会抑制局部免疫反应，可能导致真菌感染加重或扩散
- **标准处理建议**：应使用抗真菌药物，不应使用皮炎平
- **禁止表达**：脚气也能用 / 所有皮肤痒都能涂

## 不建议自行使用场景

### 场景：合并细菌或真菌感染
- **场景类型**：不建议自行使用
- **原因**：说明书明确"如并发细菌或真菌感染，请咨询医师处理"
- **标准处理建议**：咨询医师，可能需要联合抗感染治疗

### 场景：用药7日症状未缓解
- **场景类型**：不建议自行使用
- **原因**：说明书明确"用药7日后症状未缓解，应咨询医师或药师"（绿皮）；"用药1周后症状未缓解，请咨询医师"（红皮）
- **标准处理建议**：停止自行用药，就医咨询

### 场景：大面积皮损
- **场景类型**：不建议自行使用
- **原因**：大面积使用外用激素增加全身吸收风险，可能引起下丘脑-垂体-肾上腺轴抑制
- **标准处理建议**：大面积皮损应就医，由医师制定治疗方案

## 需谨慎场景

### 场景：孕妇及哺乳期妇女
- **场景类型**：需谨慎
- **原因**：绿皮和红皮说明书均明确"孕妇及哺乳期妇女慎用"
- **标准处理建议**：建议先咨询产科或皮肤科医生，不要自行使用

### 场景：婴幼儿和儿童
- **场景类型**：需谨慎
- **原因**：绿皮说明书"婴幼儿、儿童对本品更敏感，使用时应谨慎""儿童必须在成人监护下使用"；红皮说明书"小儿避免使用"
- **标准处理建议**：绿皮2岁以上可在成人监护下谨慎使用；红皮小儿应避免使用
- **禁止表达**：小孩随便涂就行

### 场景：皮肤萎缩的老年人
- **场景类型**：需谨慎
- **原因**：绿皮说明书"皮肤萎缩的老年人，对本品更敏感，故使用时应谨慎"
- **标准处理建议**：注意观察皮肤变化，如出现皮肤变薄应停药

### 场景：面部及敏感部位使用
- **场景类型**：需谨慎
- **原因**：面部、眼周、口鼻等部位皮肤薄，吸收率高，长期使用激素风险更大。说明书明确"避免接触眼睛和其他黏膜"
- **标准处理建议**：面部使用应减量、缩短疗程，避开眼周和口鼻黏膜

### 场景：运动员
- **场景类型**：需谨慎
- **原因**：绿皮和红皮说明书均注明"运动员慎用"
- **标准处理建议**：赛前使用需注意兴奋剂检测相关规定

### 场景：长期连续使用
- **场景类型**：需谨慎
- **原因**：长期大量使用可致皮肤萎缩、毛细血管扩张、色素沉着、继发感染等不良反应
- **标准处理建议**：控制使用疗程，症状缓解后应逐步减量停药
