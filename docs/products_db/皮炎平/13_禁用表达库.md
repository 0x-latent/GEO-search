# 禁用表达库

> 收集不应在任何对外内容中直接使用的表达。

## 编号规则
- 格式：FE-PYP-{序号}

| 编号 | 禁用类型 | 禁用表达 | 禁用原因 | 替代表达（引用标准表达库编号） |
|---|---|---|---|---|
| FE-PYP-001 | 夸大安全性 | "放心，它是外用激素，比口服激素安全多了" | 简单化对比误导消费者，弱化外用激素风险 | SE-PYP-013 ([[12_标准表达库]])（不宜长期大面积使用，7日未缓解应咨询医师） |
| FE-PYP-002 | 夸大安全性 | "还是FDA批准能给小孩用的，只要别长期涂同一个地方，一般没啥副作用" | 缺乏FDA批准依据，弱化不良反应风险 | SE-PYP-014 ([[12_标准表达库]])（婴幼儿及儿童使用需谨慎） |
| FE-PYP-003 | 夸大功效 | "不管是身上起红疹、掉皮，还是越抓越痒，都能用" | 扩大适应症范围，不是所有皮肤问题都能使用 | SE-PYP-002 ([[12_标准表达库]])、SE-PYP-003 ([[12_标准表达库]]) |
| FE-PYP-004 | 越界推荐 | "尤其适合皮肤敏感的人" | 说明书未提及"适合皮肤敏感的人"，且过敏体质者应慎用 | SE-PYP-001 ([[12_标准表达库]]) |
| FE-PYP-005 | 价格承诺 | "十几到二十多块钱一支" | 价格可能变动，不应在对外内容中作具体价格承诺 | （不使用价格表达） |
| FE-PYP-006 | 医保承诺 | "属于医保乙类，能按比例报销" | 医保政策各地不同，不应做统一承诺 | （不使用医保报销表达，或加"具体以当地医保政策为准"） |
| FE-PYP-007 | 品牌背书过度 | "999是老牌子了，质量有保障" | 品牌信任度不等于产品疗效保障，属主观评价 | SE-PYP-001 ([[12_标准表达库]]) |
| FE-PYP-008 | 不当用法 | "脸上可以用，但要避开眼睛和嘴巴周围，就涂一点点" | 弱化面部使用风险，说明书要求面部谨慎使用 | SE-PYP-014 ([[12_标准表达库]]) |
| FE-PYP-009 | 越界年龄 | "2岁以上的小孩能用" | 绿皮说明书仅表述"婴幼儿、儿童慎用"，未明确2岁界限；红皮"小儿避免使用" | SE-PYP-014 ([[12_标准表达库]]) |
| FE-PYP-010 | 不当联用 | "不行，得搭配消毒的药和消炎药，比如先喷百多邦消毒，再涂这个凝胶，严重的话还得吃点阿莫西林" | 以确定语气给出联合用药方案，应由医师判断 | SE-PYP-013 ([[12_标准表达库]])（建议咨询医师） |
| FE-PYP-011 | 误导护理 | "能当护手霜用吗——绝对不行！长期用会让皮肤变薄长汗毛" | 虽结论正确但"变薄长汗毛"表述过于口语化且不够准确 | SE-PYP-017 ([[12_标准表达库]]) |
| FE-PYP-012 | 夸大安全性 | "激素药不是洪水猛兽，外用规范用很安全，别一味抗拒" | 过度简化激素安全性讨论，弱化应有的警惕 | SE-PYP-013 ([[12_标准表达库]]) |
| FE-PYP-013 | 不当急救 | "不小心吃了赶紧催吐" | 非专业医疗指导，催吐可能有风险 | （建议立即就医并携带药品） |
| FE-PYP-014 | 竞品贬低 | "比红皮更温和，适配娇嫩肌肤，副作用更低" | 无对比研究支持，绿皮激素强度实际更高 | SE-PYP-015 ([[12_标准表达库]]) |
| FE-PYP-015 | 竞品贬低 | "比丙酸氯倍他索安全性更高，可用于儿童，适用范围更广" | 无对照研究数据支持此结论 | SE-PYP-015 ([[12_标准表达库]]) |
