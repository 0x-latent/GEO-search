# 产品总览

> 本模块为摘要索引，不独立维护事实。具体内容请查阅对应模块。

- 产品类别：皮肤科外用糖皮质激素类非处方药
- 产品核心功能：抗炎、抗过敏、止痒、减少渗出，用于缓解湿疹、皮炎、瘙痒等炎症性皮肤问题
- 主要适用人群/场景：湿疹、皮炎、皮肤瘙痒症患者 → [[04_核心适用场景]]
- 核心价值：两款剂型（绿皮凝胶/红皮乳膏）覆盖不同肤质和病情需求 → [[03_产品定位与核心价值]]
- 主要使用方式：局部外用，绿皮每日1次，红皮每日1-2次 → [[08_使用方法与操作规范]]
- 主要风险/限制：不得用于皮肤破溃处；孕妇哺乳期慎用；不宜长期大面积使用 → [[06_不适用禁用谨慎场景]]
- 与同类产品的主要区别：绿皮（强效）vs 红皮（中效），凝胶vs乳膏两种剂型差异 → [[09_产品差异与产品矩阵]]
- 资料来源：药品说明书（绿皮2021年修订、红皮2020年修订）、内部培训资料
