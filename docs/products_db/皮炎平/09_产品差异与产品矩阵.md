# 产品差异与产品矩阵

> 仅基于客观维度对比（成分、剂型、适用范围、使用方式、人群限制等），不做无依据的优劣判断。

## 绿皮 vs 红皮对比

| 对比维度 | 绿皮（糠酸莫米松凝胶） | 红皮（复方醋酸地塞米松乳膏） |
|---|---|---|
| 核心成分 | 糠酸莫米松（每克1mg） | 醋酸地塞米松（每克0.75mg）+ 薄荷脑（每克10mg）+ 樟脑（每克10mg） |
| 剂型/形态 | 凝胶（乳白色半透明） | 乳膏（白色，有樟脑芳香） |
| 激素强度分级 | 强效外用糖皮质激素 | 中效外用糖皮质激素 |
| 使用肤感 | 流动性好，清爽不油腻，吸收快 | 保湿效果好，封闭性强，释放持久 |
| 适应症（说明书） | 湿疹、神经性皮炎、异位性皮炎、皮肤瘙痒症 | 局限性瘙痒症、神经性皮炎、接触性皮炎、脂溢性皮炎、慢性湿疹 |
| 使用频次 | 每日1次 | 每日1～2次 |
| 适合肤质/场景 | 偏油、渗出不多、偏好清爽肤感 | 偏干、慢性反复、需更强保湿封闭 |
| 特殊禁忌 | 皮肤破溃处不得使用 | 破溃/化脓/明显渗出禁用；病毒感染（疱疹、水痘）禁用 |
| 儿童使用 | 婴幼儿、儿童慎用，须成人监护 | 小儿避免使用 |
| 规格 | 10g / 12g / 15g | 15g / 20g |
| 批准文号 | 国药准字H20080010 | 国药准字H44024170 |
| 生产企业 | 华润三九（南昌）药业有限公司 | 华润三九医药股份有限公司 |

## 选择逻辑

- 偏好清爽、吸收快，或更偏湿疹、异位性皮炎、瘙痒症场景 → 可优先考虑绿皮
- 偏好保湿、覆盖性好，或更偏慢性湿疹、接触性皮炎、脂溢性皮炎等场景 → 可优先考虑红皮
- 绿皮在激素效力上通常高于红皮，但具体使用仍需结合部位、病情、疗程和人群综合判断
