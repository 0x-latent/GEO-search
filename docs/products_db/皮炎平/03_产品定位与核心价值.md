# 产品定位与核心价值

- 产品类别定位：皮肤科外用糖皮质激素类非处方药（OTC）
- 目标问题：湿疹、皮炎（神经性皮炎、异位性皮炎、接触性皮炎、脂溢性皮炎）、皮肤瘙痒症等炎症性皮肤问题
- 核心作用：抗炎、抗过敏、止痒、减少渗出
- 核心使用场景：皮肤发红、发痒、炎症反应较明显的情况；日常轻中度皮肤炎症的自我药疗
- 主要使用人群：有湿疹、皮炎、瘙痒症状的成年人及2岁以上儿童（儿童需在成人监护下使用）
- 主要差异点：双剂型策略（绿皮凝胶清爽型 + 红皮乳膏滋润型），覆盖不同肤质和病情偏好；999为知名OTC品牌，消费者认知度高
- 不应承诺的内容：
  - 不应承诺"根治"湿疹、皮炎等慢性皮肤病
  - 不应承诺"无副作用""绝对安全"
  - 不应承诺适用于所有皮肤问题（如真菌感染、病毒感染不适用）
  - 不应承诺可替代就医诊断
