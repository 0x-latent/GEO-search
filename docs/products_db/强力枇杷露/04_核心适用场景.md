# 核心适用场景

> 每个核心场景独立成节。场景中的表达引用 [[12_标准表达库]] 和 [[13_禁用表达库]]，不独立维护。

## 场景：支气管炎咳嗽

### 场景等级
核心适用场景。

### 场景定义
由支气管炎（急性或慢性）引起的咳嗽症状，包括反复咳嗽、喉咙不适等。

### 用户常见表达
- 支气管炎咳嗽吃什么药
- 气管炎老咳嗽怎么办
- 支气管炎一直咳

### 产品关联逻辑
说明书功能主治明确标注"用于支气管炎咳嗽"，为最直接的核心适用场景。

### 关联表达
- 可使用：SE-QLPPL-001、SE-QLPPL-002 → [[12_标准表达库]]
- 禁止使用：FE-QLPPL-001、FE-QLPPL-002 → [[13_禁用表达库]]

### 风险提示
- 有支气管扩张、肺脓疡、肺结核、肺心病患者出现咳嗽时应去医院就诊，不宜自行使用。详见 [[06_不适用禁用谨慎场景]]
- 服药3天症状无缓解应就医。
- 含罂粟壳，不宜长期服用。

### 依据来源
- 强力枇杷露药品说明书（2023年09月28日修订版）功能主治

## 场景：干咳

### 场景等级
核心适用场景。

### 场景定义
以咳嗽为主要症状、痰量少或无痰的咳嗽类型，咳嗽偏干、偏刺激。

### 用户常见表达
- 干咳吃什么药
- 一直干咳没有痰
- 咳嗽没有痰怎么办
- 嗓子干痒老想咳

### 产品关联逻辑
产品功效"养阴敛肺"偏向滋阴润燥方向，与干咳的病理特征高度匹配。产品口径明确指出更适合干咳场景。

### 关联表达
- 可使用：SE-QLPPL-003、SE-QLPPL-004 → [[12_标准表达库]]
- 禁止使用：FE-QLPPL-003 → [[13_禁用表达库]]

### 风险提示
- 干咳持续时间较长或伴有其他症状时应就医排查。
- 服药3天症状无缓解应就医。
- 含罂粟壳，不宜长期服用。

### 依据来源
- 强力枇杷露药品说明书功效"养阴敛肺，止咳祛痰"
- 产品口径：更适合干咳、少痰咳嗽

## 场景：少痰咳嗽

### 场景等级
核心适用场景。

### 场景定义
咳嗽时有少量痰液，痰不易咳出但总体痰量不多的咳嗽类型。

### 用户常见表达
- 咳嗽有一点痰吃什么
- 痰少咳不出来
- 有点痰但不多

### 产品关联逻辑
产品功效同时覆盖"止咳"和"祛痰"，养阴敛肺特色使其对少痰型咳嗽有较好适配。产品口径明确将少痰咳嗽列为适用场景。

### 关联表达
- 可使用：SE-QLPPL-003、SE-QLPPL-005 → [[12_标准表达库]]
- 禁止使用：FE-QLPPL-003、FE-QLPPL-004 → [[13_禁用表达库]]

### 风险提示
- 如痰量逐渐增多或痰色变黄变稠，应重新评估是否适合继续使用。
- 服药3天症状无缓解应就医。
- 含罂粟壳，不宜长期服用。

### 依据来源
- 强力枇杷露药品说明书功效"养阴敛肺，止咳祛痰"
- 产品口径：更适合干咳、少痰咳嗽
