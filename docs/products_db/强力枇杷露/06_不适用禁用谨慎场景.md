# 不适用/禁用/需谨慎场景

> 分为禁用场景、不建议自行使用场景、需谨慎场景三类。

## 禁用场景

### 场景：儿童使用
- **场景类型**：禁用
- **原因**：说明书明确标注"儿童禁用"
- **标准处理建议**：明确告知儿童禁用，建议咨询儿科医生选择适合儿童的止咳药物
- **禁止表达**：小孩也可以喝 / 减量给孩子用 / 儿童少喝一点没关系

### 场景：孕妇使用
- **场景类型**：禁用
- **原因**：说明书明确标注"孕妇禁用"。产品含罂粟壳，对胎儿有潜在风险
- **标准处理建议**：明确告知孕妇禁用，建议咨询产科或呼吸科医生
- **禁止表达**：孕妇可以少量服用 / 怀孕了也能喝

### 场景：哺乳期妇女使用
- **场景类型**：禁用
- **原因**：说明书明确标注"哺乳期妇女禁用"。产品含罂粟壳，成分可能通过乳汁影响婴儿
- **标准处理建议**：明确告知哺乳期妇女禁用，建议咨询医生选择哺乳期安全的止咳药物
- **禁止表达**：喂奶期间也可以喝 / 哺乳期用量减半就行

### 场景：糖尿病患者使用
- **场景类型**：禁用
- **原因**：说明书明确标注"糖尿病患者禁服"。产品辅料含单糖浆、蔗糖
- **标准处理建议**：明确告知糖尿病患者禁服，建议咨询医生选择无糖型止咳药物
- **禁止表达**：糖尿病也可以喝一点 / 含糖不多影响不大

### 场景：对本品及所含成分过敏者
- **场景类型**：禁用
- **原因**：说明书明确标注"对本品及所含成分过敏者禁用"。说明书记录有过敏性休克个案报告
- **标准处理建议**：告知过敏者禁用，如有过敏史应避免使用
- **禁止表达**：过敏体质也没事 / 试试看会不会过敏

## 不建议自行使用场景

### 场景：支气管扩张患者咳嗽
- **场景类型**：不建议自行使用
- **原因**：说明书注意事项明确要求"有支气管扩张患者出现咳嗽时应去医院就诊"
- **标准处理建议**：建议就医，由医生判断是否适合使用

### 场景：痰多痰黄稠厚型咳嗽
- **场景类型**：不建议自行使用
- **原因**：产品定位偏向干咳、少痰场景，养阴敛肺功效不适合痰多痰稠的情况。敛肺可能不利于痰液排出
- **标准处理建议**：建议选择更适合痰多型咳嗽的祛痰药物，或就医咨询

### 场景：长期服用
- **场景类型**：不建议自行使用
- **原因**：说明书明确标注"本品含罂粟壳，不宜长期服用"。有过量服用导致药物依赖的个案报告
- **标准处理建议**：告知不宜长期服用，服药3天症状无缓解应就医

## 需谨慎场景

### 场景：严重高血压患者使用
- **场景类型**：需谨慎
- **原因**：说明书标注"严重高血压患者应在医师指导下服用"
- **标准处理建议**：建议在医师指导下使用

### 场景：年老体弱者使用
- **场景类型**：需谨慎
- **原因**：说明书标注"年老体弱者应在医师指导下服用"
- **标准处理建议**：建议在医师指导下使用

### 场景：过敏体质者使用
- **场景类型**：需谨慎
- **原因**：说明书标注"过敏体质者慎用"
- **标准处理建议**：建议谨慎使用，如出现过敏反应应立即停药

### 场景：运动员使用
- **场景类型**：需谨慎
- **原因**：说明书标注"运动员慎用"。产品含罂粟壳，可能导致药检阳性
- **标准处理建议**：建议运动员避免使用或在专业人员指导下使用

### 场景：驾驶或操作精密仪器者使用
- **场景类型**：需谨慎
- **原因**：说明书标注"驾驶员、车、船、从事精密作业、机械作业及操作精密仪器者慎用"
- **标准处理建议**：建议使用期间避免驾驶或操作精密仪器
