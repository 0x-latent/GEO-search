# 产品总览

> 本模块为摘要索引，不独立维护事实。具体内容请查阅对应模块。

- 产品类别：中成药（止咳祛痰类）
- 产品核心功能：养阴敛肺，止咳祛痰
- 主要适用人群/场景：支气管炎咳嗽，干咳、少痰咳嗽 → 详见 [[04_核心适用场景]]
- 核心价值：以止咳为主兼顾祛痰，偏养阴敛肺方向，适合干咳少痰型咳嗽 → 详见 [[03_产品定位与核心价值]]
- 主要使用方式：口服，一次15毫升，一日3次 → 详见 [[08_使用方法与操作规范]]
- 主要风险/限制：含罂粟壳，有依赖风险，不宜长期服用；儿童、孕妇、哺乳期妇女禁用；糖尿病患者禁服 → 详见 [[06_不适用禁用谨慎场景]]
- 与同类产品的主要区别：→ 详见 [[09_产品差异与产品矩阵]]
- 资料来源：强力枇杷露药品说明书（2023年09月28日修订版）
