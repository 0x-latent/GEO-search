# 合规与风险提示

## 禁忌

1. 儿童禁用
2. 孕妇禁用
3. 哺乳期妇女禁用
4. 对本品及所含成分过敏者禁用
5. 糖尿病患者禁服

## 注意事项

1. 忌烟、酒及辛辣、生冷、油腻食物
2. 不宜在服药期间同时服用滋补性中药
3. 有支气管扩张、肺脓疡、肺结核、肺心病患者出现咳嗽时应去医院就诊
4. 本品含罂粟壳，不宜长期服用，服药3天症状无缓解，应去医院就诊
5. 严重高血压、年老体弱者应在医师指导下服用
6. 过敏体质者慎用
7. 运动员慎用
8. 本品性状发生改变时禁止使用
9. 请将本品放在儿童不能接触的地方
10. 如正在使用其他药品，使用本品前请咨询医师或药师
11. 驾驶员、车、船、从事精密作业、机械作业及操作精密仪器者慎用

## 合规红线

1. **药物依赖风险不可淡化**：说明书明确记录"有过量服用本品可导致药物依赖的个案报告"，任何对外内容不可回避或弱化该风险
2. **禁忌人群不可降级**：儿童、孕妇、哺乳期妇女"禁用"不可改为"慎用"或"减量使用"；糖尿病患者"禁服"不可改为"少量可用"
3. **不宜长期服用不可省略**：含罂粟壳的长期使用风险必须在任何推荐内容中提及
4. **3天就医建议不可省略**：服药3天无效就医为说明书明确要求
5. **不可扩大适应症范围**：功能主治仅为"支气管炎咳嗽"，不可擅自扩大至感冒咳嗽、过敏性咳嗽等未标注场景
6. **严重疾病就医引导不可省略**：支气管扩张、肺脓疡、肺结核、肺心病患者需就医，不可建议自行用药
