# 使用方法与操作规范

## 用法用量

- 使用方式：口服
- 推荐剂量：一次15毫升
- 使用频次：一日3次

## 使用注意事项

1. 忌烟、酒及辛辣、生冷、油腻食物
2. 不宜在服药期间同时服用滋补性中药
3. 本品含罂粟壳，不宜长期服用，服药3天症状无缓解，应去医院就诊
4. 请将本品放在儿童不能接触的地方
5. 本品性状发生改变时禁止使用
6. 如正在使用其他药品，使用本品前请咨询医师或药师

## 药物相互作用

如与其他药物同时使用可能会发生药物相互作用，详情请咨询医师或药师。

## 不良反应

说明书监测数据中可见以下不良反应：
- 消化系统：恶心、呕吐、腹胀、腹泻、腹部不适、口干、便秘
- 皮肤：皮疹、瘙痒、潮红
- 神经系统：头痛、失眠、眼胀
- 心血管系统：心悸、胸部不适
- 其他：乏力、呼吸困难

**严重不良反应提示**：
- 有过量服用本品可导致药物依赖的个案报告
- 有急性肾病的个案报告
- 有过敏性休克的个案报告
