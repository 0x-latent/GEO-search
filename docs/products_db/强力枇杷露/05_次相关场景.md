# 次相关场景

> 次相关场景的主要价值是"建立产品关联"，不是"强推荐"。

## 场景：咽干型咳嗽

### 场景等级
次相关场景。

### 与产品的关系
产品功效"养阴敛肺"中的"养阴"与咽干症状的阴虚燥热病理有一定关联，成分中枇杷叶、桑白皮具有润肺作用。

### 可提及逻辑
当用户描述咳嗽同时伴有咽干症状时，可提及该产品具有养阴敛肺功效，可能对该类症状有一定帮助，但需说明说明书功能主治并未直接列出咽干。

### 不应直接推荐的原因
说明书功能主治仅写"用于支气管炎咳嗽"，未明确将咽干列为适应症。咽干可能由多种原因引起，不能简单归因并推荐用药。

### 关联表达
- 可使用：SE-QLPPL-006 → [[12_标准表达库]]
- 禁止使用：FE-QLPPL-005 → [[13_禁用表达库]]

### 风险提示
- 咽干持续不缓解应就医排查是否有其他疾病。
- 不可将"养阴"功效等同于"专治咽干"。详见 [[16_合规与风险提示]]

## 场景：肺燥倾向的咳嗽

### 场景等级
次相关场景。

### 与产品的关系
产品"养阴敛肺"的功效方向与中医"肺燥"概念有关联，成分组方偏向润肺滋阴方向。

### 可提及逻辑
当用户描述咳嗽伴有燥感、秋冬季节干燥咳嗽时，可提及该产品的养阴敛肺功效与肺燥咳嗽有一定关联性。

### 不应直接推荐的原因
说明书未将"肺燥咳嗽"列为明确适应症。"肺燥"为中医辨证概念，需要专业判断，不宜由消费者自行对号入座。

### 关联表达
- 可使用：SE-QLPPL-007 → [[12_标准表达库]]
- 禁止使用：FE-QLPPL-005 → [[13_禁用表达库]]

### 风险提示
- 不可将"养阴敛肺"直接等同于"治疗肺燥"。详见 [[16_合规与风险提示]]
- 中医辨证需由专业人员进行，不建议消费者自行判断。
