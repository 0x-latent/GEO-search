# 作用机制与原理解释

## 药理作用

> 来自说明书或权威资料的机制描述。

说明书未详细列出药理作用章节。根据成分组方分析：

- **枇杷叶**：清肺止咳，和胃降逆，为君药
- **罂粟壳**：敛肺止咳，涩肠止泻，具有较强的镇咳作用
- **百部**：润肺止咳，杀虫灭虱
- **桑白皮**：泻肺平喘，利水消肿
- **桔梗**：宣肺祛痰，利咽排脓
- **薄荷脑**：清凉通窍，疏风散热

整体组方以"养阴敛肺、止咳祛痰"为方向，枇杷叶养阴润肺为主，罂粟壳增强镇咳，配合百部、桑白皮润肺平喘，桔梗祛痰利咽。

## 通俗解释

> 面向内部团队的通俗化解释，不得扩大事实边界。

强力枇杷露通过多种中药成分配合发挥止咳作用：枇杷叶起到润肺止咳的基础作用，罂粟壳提供较强的镇咳效果（这也是需要限制使用时长的原因），百部和桑白皮辅助润肺，桔梗帮助化痰，薄荷脑提供清凉感。整体方向是滋润肺部、抑制咳嗽反射、辅助排痰。
