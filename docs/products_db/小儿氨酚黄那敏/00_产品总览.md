# 产品总览

> 本模块为摘要索引，不独立维护事实。具体内容请查阅对应模块。

- 产品类别：儿童复方感冒药（OTC）
- 产品核心功能：缓解儿童普通感冒及流行性感冒引起的发热、头痛、鼻塞、流涕、打喷嚏等症状
- 主要适用人群/场景：儿童感冒发热、头痛、鼻塞、流涕、打喷嚏 → [[04_核心适用场景]]
- 核心价值：三重成分（退热+抗过敏+清热）协同缓解儿童感冒多种症状 → [[03_产品定位与核心价值]]
- 主要使用方式：温开水冲服，按年龄体重给药 → [[08_使用方法与操作规范]]
- 主要风险/限制：对成分过敏者禁用；严重肝肾功能不全者慎用；不能与其他含对乙酰氨基酚药物同服 → [[06_不适用禁用谨慎场景]]
