# 不适用/禁用/需谨慎场景

> 分为禁用场景、不建议自行使用场景、需谨慎场景三类。

## 禁用场景

### 场景：对本品成分过敏者
- **场景类型**：禁用
- **原因**：对对乙酰氨基酚、马来酸氯苯那敏或人工牛黄过敏者使用本品可能引发过敏反应
- **标准处理建议**：明确告知禁用，建议就医由医生选择替代药物
- **禁止表达**：可放心使用 / 没问题 / 直接用即可

### 场景：正在服用其他含对乙酰氨基酚药物
- **场景类型**：禁用
- **原因**：对乙酰氨基酚重复用药可导致剂量过大，增加肝脏损伤风险
- **标准处理建议**：明确告知不可同服，需停用其中一种或咨询医生调整用药方案
- **禁止表达**：一起吃没事 / 可以叠加使用

## 不建议自行使用场景

### 场景：婴幼儿及特殊基础疾病儿童
- **场景类型**：不建议自行使用
- **原因**：婴幼儿用药安全性要求更高，有基础疾病的儿童可能存在药物相互作用或禁忌
- **标准处理建议**：建议在医生指导下使用，不可自行判断剂量

### 场景：高热持续不退或伴有异常表现
- **场景类型**：不建议自行使用
- **原因**：持续高热、精神差、抽搐、呼吸异常、反复呕吐等可能提示严重感染或其他疾病，不应仅依赖对症药物
- **标准处理建议**：建议尽快就医，不能只依赖自行用药

## 需谨慎场景

### 场景：严重肝肾功能不全者
- **场景类型**：需谨慎
- **原因**：对乙酰氨基酚主要经肝脏代谢，严重肝肾功能不全者使用可能加重肝肾负担
- **标准处理建议**：慎用或遵医嘱，建议在医生指导下决定是否使用及剂量调整

### 场景：长期用药
- **场景类型**：需谨慎
- **原因**：连续用药后症状无改善或症状加重，提示需要重新评估病情
- **标准处理建议**：连续用药后症状无改善应及时就医
