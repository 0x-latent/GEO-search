# 使用方法与操作规范

## 用法用量

- 使用方式：温开水冲服
- 推荐剂量：按年龄或体重给药，应严格对应说明书剂量，不可自行加量（具体分年龄/体重剂量表待补充）
- 使用频次：待补充（以说明书为准）

## 使用注意事项

1. 严格按照儿童年龄和体重使用，不可超量、超频次服用
2. 不要与其他含对乙酰氨基酚的药物同服，以免对乙酰氨基酚过量
3. 对本品任一成分过敏者禁用
4. 如有肝功能问题、肾功能问题或长期服药情况，应在医生指导下使用
5. 连续用药后症状无改善或症状加重，应及时就医
6. 发热明显升高、持续不退，或孩子精神差、食欲极差时，不应只依赖自行用药

## 药物相互作用

- 不可与其他含对乙酰氨基酚的感冒药重复使用（对乙酰氨基酚过量有肝损伤风险）
