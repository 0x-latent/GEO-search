# 产品总览

> 本模块为摘要索引，不独立维护事实。具体内容请查阅对应模块。

- 产品类别：中西药复方感冒药（OTC）
- 产品核心功能：解热镇痛，缓解普通感冒引起的头痛、发热、鼻塞、流涕、咽痛
- 主要适用人群/场景：普通感冒患者 → 详见 [[04_核心适用场景]]
- 核心价值：中西药复方，兼顾中药清热与西药退热镇痛 → 详见 [[03_产品定位与核心价值]]
- 主要使用方式：开水冲服，一次10g，一日3次 → 详见 [[08_使用方法与操作规范]]
- 主要风险/限制：严重肝肾功能不全者禁用；不能与其他含对乙酰氨基酚药物同服 → 详见 [[06_不适用禁用谨慎场景]]
- 与同类产品的主要区别：待补充 → 详见 [[09_产品差异与产品矩阵]]
