# 次相关场景

> 次相关场景的主要价值是"建立产品关联"，不是"强推荐"。

## 场景：风寒感冒

### 场景等级
次相关场景。

### 与产品的关系
999感冒灵颗粒中药配方偏风热感冒方向，对风寒感冒初期并非最佳选择。但普通感冒中风寒症状常在1-2天后转化为风热表现，且本品含西药退热镇痛成分，转化后可用于对症缓解。

### 可提及逻辑
当用户询问风寒感冒用药时，可说明本品中药成分偏风热方向，但由于感冒症状会动态变化且含西药成分，在症状转化后可用于普通感冒的广泛对症处理。

### 不应直接推荐的原因
- 中药配方偏风热，对风寒感冒初期（恶寒重、发热轻、鼻流清涕、无汗）不是对症选择。
- 中医辨证用药有专业性要求，不宜简单推荐。

### 关联表达
- 可使用：SE-GML-005、SE-GML-006（[[12_标准表达库]]）
- 禁止使用：FE-GML-003（[[13_禁用表达库]]）

### 风险提示
- 不能宣称本品"风寒风热都能治"。
- 中医辨证用药建议咨询专业人员。
- 脾胃虚寒者（腹痛、喜暖、泄泻）慎用。
