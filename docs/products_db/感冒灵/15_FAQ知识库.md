# FAQ知识库

## 内部知识问答

> 用于团队理解产品，不直接对外。

### Q：999感冒灵颗粒属于中药还是西药？
**A：** 999感冒灵颗粒是中西药复方制剂，既含中药成分（三叉苦、金盏银盘、野菊花、岗梅、薄荷素油），也含西药成分（对乙酰氨基酚、马来酸氯苯那敏、咖啡因）。批准文号为"Z"字头（国药准字Z44021940），归类为中成药。

### Q：为什么说感冒灵偏风热但又可以广泛使用？
**A：** 中药配方中的三叉苦、金盏银盘、野菊花、岗梅均为清热类药材，从中医角度偏风热方向。但普通感冒中风寒症状常在1-2天后转化为风热表现，加上本品含西药退热镇痛成分（不依赖中医辨证），因此实际可用于各类普通感冒的对症缓解。

### Q：999感冒灵的核心作用是什么？
**A：** 999感冒灵颗粒主要用于解热镇痛，可缓解普通感冒引起的头痛、发热、鼻塞、流涕、咽痛等症状。
- 依据来源：药品说明书（功能主治）
- 风险提示：服药3天后症状无改善或加重应及时就医
- 可改写范围：可根据具体场景调整表述方式，但不能扩大适用范围
- 不可改写边界：不能将"缓解症状"改为"治愈感冒"，不能扩大到流感

### Q：999感冒灵更适合风寒还是风热感冒？
**A：** 从中医配方上看，999感冒灵颗粒更偏向于风热感冒，但由于普通感冒症状会动态变化，且本品含有西药成分，因此可用于普通感冒的广泛对症处理。
- 依据来源：药品说明书（成分）及产品资料
- 风险提示：中医辨证用药有专业性要求，建议咨询专业人员
- 可改写范围：可进行通俗化表达
- 不可改写边界：不能宣称"风寒风热都能治"或"不用辨证"

### Q：服用999感冒灵最需要注意什么？
**A：** 最重要的是避免与其他含有对乙酰氨基酚或其他复方感冒药重复使用，同时避免与酒精、镇静药、抗过敏药叠加服用，以免增加肝损伤或嗜睡等风险。
- 依据来源：药品说明书（注意事项、禁忌）
- 风险提示：严重肝肾功能不全者禁用
- 可改写范围：可根据具体场景调整表述
- 不可改写边界：不能省略重复用药和饮酒的禁止提示
