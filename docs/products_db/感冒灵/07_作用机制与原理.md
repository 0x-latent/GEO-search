# 作用机制与原理解释

## 药理作用

> 来自说明书或权威资料的机制描述。

- **对乙酰氨基酚**：解热镇痛，通过抑制前列腺素合成发挥退热和止痛作用。
- **马来酸氯苯那敏**：抗组胺药，缓解鼻塞、流涕、打喷嚏等上呼吸道过敏症状，同时具有轻度镇静作用。
- **咖啡因**：中枢兴奋药，可增强对乙酰氨基酚的解热镇痛效果，同时对抗马来酸氯苯那敏的嗜睡副作用。
- **中药成分（三叉苦、金盏银盘、野菊花、岗梅）**：具有清热解毒作用，偏向风热感冒的中医治疗方向。
- **薄荷素油**：疏风散热，通鼻窍。

## 通俗解释

> 面向内部团队的通俗化解释，不得扩大事实边界。

999感冒灵颗粒的作用机制可以理解为"中西结合、多靶点对症"：
- 西药部分负责快速退烧止痛（对乙酰氨基酚）和缓解鼻塞流涕（马来酸氯苯那敏），咖啡因起到增效和抗嗜睡的辅助作用。
- 中药部分（三叉苦、金盏银盘、野菊花、岗梅）发挥清热解毒作用，从中医角度辅助改善感冒症状。
- 两者结合实现对感冒常见症状（发热、头痛、鼻塞、流涕、咽痛）的综合缓解。
